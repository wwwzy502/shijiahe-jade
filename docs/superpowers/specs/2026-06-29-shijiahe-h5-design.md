# 玉见·石家河 — H5交互作品设计文档

## 项目概述

- **作品名称**：玉见·石家河
- **主题**：湖北天门石家河文化虚拟考古互动体验
- **形式**：原生 HTML5 单页式交互作品（6场景）
- **核心隐喻**：用户扮演考古学家，完成"发掘→鉴赏→复原→回顾→传承"的完整体验
- **技术栈**：HTML + CSS + JavaScript + Canvas + GSAP
- **兼容目标**：移动端浏览器（微信内置浏览器、Safari、Chrome），响应式适配 375-768px

## 场景设计

### 场景一：序幕「玉见」

- **目的**：第一印象，营造神秘氛围
- **布局**：垂直居中，玉凤剪影（SVG）+ 标题 + 副标题 + CTA按钮
- **交互**：点击「开启探索」→ 过渡到场景二
- **背景**：Canvas 粒子缓慢浮动

### 场景二：考古发掘「拂去尘埃」

- **目的**：核心交互，模拟考古发掘
- **布局**：Canvas 刮擦区域 + 进度条 + 文物信息
- **交互**：手指滑动擦除泥土层（globalCompositeOperation: destination-out），露出底层文物图
- **阈值**：擦除60%后激活「查看文物」按钮
- **首发文物**：玉虎

### 场景三：文物鉴赏「凝视为真」

- **目的**：文物细节欣赏
- **布局**：文物展示区 + 缩略图栏 + 信息卡片
- **交互**：拖拽切换多角度图片（6-8帧序列），模拟3D旋转；左右滑动切换文物
- **文物列表**：玉虎、玉凤、玉人首、玉蝉等
- **额外**：放大镜查看纹饰细节

### 场景四：祭祀复原「礼敬天地」

- **目的**：理解先民祭祀文化
- **布局**：祭祀场景底图（含虚线目标位）+ 底部拖拽元素栏
- **交互**：拖拽玉琮/玉璧/祭司/火堆等到正确位置；全部归位后场景"激活"（动画）
- **反馈**：正确=落入+光晕，错误=弹回+提示

### 场景五：时光长廊「七秩寻踪」

- **目的**：了解70年考古历程
- **布局**：横向滚动时间轴 + 详情卡片
- **交互**：惯性滑动 + 节点吸附居中 + 详情联动
- **节点**：1954首次发现、1987肖家屋脊、1990罗家柏岭、2015谭家岭、2024七十周年

### 场景六：尾声「文明的回响」

- **目的**：情感收束 + 社交分享
- **布局**：玉凤完整呈现 + 考古发现证书 + 操作按钮
- **交互**：html2canvas 生成证书截图保存；「再次探索」回到场景一
- **动态数据**：证书内容根据用户行为填充

## 导航系统

- 底部常驻：页面指示器（圆点）+ 返回/下一页
- 过渡动画：淡入淡出 + 上移，400ms ease-out
- 每场景一屏（100vh），禁止默认滚动

---

## 交互形式汇总（≥3种）

1. **Canvas 刮擦交互**（场景二）— 模拟考古发掘
2. **拖拽旋转/序列帧**（场景三）— 文物3D旋转鉴赏
3. **拖拽拼合**（场景四）— 祭祀场景元素归位
4. **横向惯性滑动**（场景五）— 时间轴浏览
5. **证书截图保存**（场景六）— html2canvas 社交分享
